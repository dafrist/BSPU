Что такое Python?

Python — это мощный и простой язык программирования, который отлично подходит для начинающих. Его лаконичный синтаксис помогает легко изучить основы программирования, при этом язык обладает достаточной мощностью для разработки серьезных проектов, таких как веб-приложения, игры, автоматизация задач и даже искусственный интеллект.

Первый пример кода на Python выглядит очень просто:

print("Привет, мир!")

Эта короткая программа выводит надпись «Привет, мир!» на экран. С помощью Python вы сможете написать свои первые программы уже через несколько уроков!

Переменные и типы данных

Переменные похожи на контейнеры, в которых можно хранить различную информацию. Python работает с такими типами данных, как:

  • int — целые числа (например, 15, -8)
  • float — дробные числа (например, 3.14, 1.618)
  • str — текстовые строки (например, «Python», «школа»)
  • bool — логические значения (True или False)

Вот пример объявления переменных и их использования:

name = "Аня"
age = 17
is_student = True
print(f"Имя: {name}, Возраст: {age}, Учится в школе: {is_student}")

В Python не нужно явно указывать тип переменной — он определяется автоматически. Это делает код простым и гибким.

Условия и логика в Python

Часто нужно, чтобы программа могла принимать решения в зависимости от условий. Для этого используются конструкции if, elif и else.

Например, проверим возраст:

age = int(input("Сколько тебе лет? "))
if age >= 18:
    print("Ты уже взрослый!")
elif age >= 13:
    print("Ты подросток")
else:
    print("Ты еще ребенок")

Эта простая программа задает вопрос и выводит разные сообщения в зависимости от введённого возраста.

Циклы для повторения действий

Циклы позволяют выполнять действия многократно. В Python есть два основных вида циклов: for и while.

Цикл for удобно использовать, если заранее известно количество повторений:

for i in range(5):
    print(f"Это повторение номер {i+1}")

Цикл while используется, когда повторения зависят от условий:

count = 0
while count < 5:
    print(f"Цикл повторился {count + 1} раз")
    count += 1

Циклы делают ваш код короче и удобнее для решения повторяющихся задач!
